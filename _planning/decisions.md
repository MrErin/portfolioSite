# Decisions

| # | Date | Decision | Rationale |
|---|------|----------|-----------|
| 1 | 2026-07-03 | Reversed whimsy slider polarity: 0=min whimsy (left), 2=max whimsy (right) | Standard UX convention is left-to-right = less-to-more. All consumers use config flags, not raw level numbers, so the change was contained to 3 files. |
| 2 | 2026-07-03 | Deferred SVG ornamentation for slider (filigree thumb, decorative rail arrows) | Pure CSS hit its ceiling for ornate detail. SVG assets should be created alongside Victorian frame overlays (Phase 3) for visual consistency. |
| 3 | 2026-07-04 | Added collapse/expand behavior to WhimsySlider (auto-collapse after 3s, re-collapse 2s after interaction, keyboard focus re-expands) | Slider occupies fixed screen space and competes with content. Collapsing to a small pill keeps it accessible without visual clutter. Keyboard users can still reach it via Tab (onFocus re-expands). |
| 4 | 2026-07-04 | Stop 2 cave floor placed as static sibling after the parallax `<section>`, not scroll-linked inside the sticky container | Scroll-linked opacity requires `useTransform` wiring and still has a visible gap after the last card. Static placement is simpler and achieves the same "cave floor appears at the end" effect. Blank scroll gap is a known tradeoff (deferred item #4). **Reversed in Phase 6:** Floor moved inside sticky container with `useTransform([0.78, 0.95])` translateY to eliminate dead scroll space. |
| 5 | 2026-07-04 | Scroll arrow placed at top of ProjectsSection parallax section rather than in Hero | Positioning the arrow immediately above the content being scrolled is clearer UX than placing it in the hero above the fold. Used `arrow.svg` with inline `filter: invert(1)` style (accepted inconsistency with `.falling-object` CSS class pattern). |
| 6 | 2026-07-04 | Stop 0 hero: shrink to 30vh + remove subheading instead of adding `<hr>` divider | The visual goal (professional, non-ornate separator) is achieved by the compact height and title change. Fewer DOM elements. Old "Falling down the rabbit hole..." subheading and gradient divider removed in the same pass. |
| 7 | 2026-07-04 | Grid mode: heading `mb-20` + card container `mt-12` creates 128px combined gap | Extra spacing was reviewed and accepted as part of the visual polish pass. |
