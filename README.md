# Portfolio Site
Code for my portfolio site. The main feature is my projects page, featuring several complicated visual effects and a whimsy modifier. On the highest level of whimsy, you get "falling down the rabbit hole" visuals, and at the lowest, you get a simple grid with a list of my projects. It was very complicated to envision and execute but also a LOT of fun.


## Tech Stack

- **React 18** + **TypeScript** - UI framework with type safety
- **Vite** - Fast build tool and dev server
- **Tailwind CSS v4** - Utility-first CSS with custom dark theme
- **Framer Motion** - Smooth animations and transitions
- **React Icons** - Icon library for navigation buttons
